# MedLex V-index

의학 용어의 접두사와 접미사를 체계적으로 색인, 검색, 관리, 학습할 수 있도록 설계된
데이터 분석 및 학습용 단일 파일 웹 애플리케이션입니다.

---

## 바로 시작하기

별도 설치 없이 아래 링크에서 바로 사용할 수 있습니다.

https://linuse765.github.io/MedLex.github.io/

1. 상단 단어장 탭에서 단어와 뜻을 추가합니다.
2. 폴더를 만들어 계통별로 분류합니다.
3. 학습 탭에서 외우기 또는 퀴즈 모드로 학습을 시작합니다.
4. 통계 탭에서 학습 현황과 취약 단어를 확인합니다.

---

## 핵심 기능

### 1. 의학 용어 접사 검색 및 필터링
- 접두사와 접미사를 시각적으로 구분하여 탐색할 수 있습니다.
- 검색창 입력 값에 동기화되어 일치하는 단어와 의미를 실시간으로 필터링합니다.

### 2. 카테고리 기반 데이터 관리
- 의학 계통별 또는 학습 목적별로 단어를 분류하는 폴더 구조를 제공합니다.
- 생성한 폴더 목록이 선택 셀렉터에 실시간으로 반영됩니다.

### 3. 사용자 커스텀 데이터 확장
- 필요한 의학 용어와 뜻을 직접 입력하여 데이터베이스를 확장할 수 있습니다.
- 필수 값 누락이나 중복 입력을 방지하는 유효성 검증이 적용되어 있습니다.

### 4. 적응형 학습 알고리즘

단순 반복이 아닌, 사용자의 실제 기억 상태를 모델링하여 최적의 단어를 출제합니다.

- **베이지안 지식 추적**: 정답과 오답 이력을 바탕으로 "이 단어를 알고 있을 확률"을
  실시간으로 갱신합니다. 확률이 95% 이상이면 마스터로 간주하고 출제 빈도를 낮춥니다.

- **기억 반감기 기반 출제 순서 조정**: 응답 속도를 추적하여 기억의 반감기를 계산합니다.
  빠르게 답할수록 오래 기억하는 것으로, 느리게 답할수록 망각 위험이 높은 것으로 판단해
  우선 출제합니다.

- **스마트 모드**: 두 알고리즘을 결합하여 현재 가장 복습이 필요한 단어를
  우선적으로 출제하는 동적 출제 순서를 생성합니다.

### 5. 실시간 통계 분석 대시보드
- 전체 통계, 접두사/접미사 비율, 폴더별 분포 등을 시각적으로 확인할 수 있습니다.
- 학습 연속 일수, 지식 확률 추이, 취약 단어 상위 5개를 제공합니다.

### 6. 학습 로그
- 학습 중 메모나 피드백을 기록할 수 있는 인앱 로그 기능을 제공합니다.
- 기록은 날짜와 시각이 함께 저장되며 최신순으로 정렬됩니다.

## 데이터 저장 방식과 한계

이 앱은 브라우저의 로컬 스토리지를 사용합니다. 별도 서버나 계정 없이 데이터가 기기에
직접 저장되므로 인터넷 연결 없이도 동작하지만, 아래 사항을 반드시 확인하세요.

- 브라우저 데이터를 초기화하면 모든 학습 데이터가 삭제됩니다.
- 기기나 브라우저를 바꾸면 데이터가 이전되지 않습니다.
- 설정 탭의 내보내기 기능으로 데이터를 JSON 파일로 백업할 수 있습니다.
- 중요한 데이터는 주기적으로 내보내기를 통해 보관하는 것을 권장합니다.

## 기술 스펙

- 외부 라이브러리나 프레임워크 의존성이 없는 순수 HTML5, CSS3, Vanilla JavaScript 구조입니다.
- 단일 파일 내에서 상태 기반 UI 렌더링이 이루어집니다.
- 미디어 쿼리를 통한 반응형 웹 디자인이 적용되어 있습니다.
- CSS 변수 체계를 기반으로 전역 라이트/다크 테마 전환이 가능합니다.
- 렌더링 부하가 큰 웹 필터 연산을 배제하고 GPU 가속 레이어를 활용하는
  transform 속성 위주로 애니메이션을 제어합니다.
